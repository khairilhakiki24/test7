Manual Book KasPro

Panduan Lengkap Penggunaan, Instalasi & Modifikasi Web App PWA

1 Panduan Penggunaan Utama

KasPro adalah aplikasi 100% Offline PWA untuk mengelola uang kas, iuran, dan patungan kelas/organisasi secara profesional. Data disimpan langsung di browser perangkat Anda (IndexedDB).

  • Tambah Anggota: Pergi ke menu Anggota, klik tombol Tambah. Anda harus membuat anggota dulu sebelum menerima iuran.
  • Mencatat Iuran (Pemasukan): Klik Terima Iuran di Beranda atau di menu Transaksi. Pilih anggota, masukkan nominal dan keterangan. Data draf otomatis tersimpan jika Anda tidak sengaja keluar/refresh web.
  • Mencatat Pengeluaran: Klik Catat Pengeluaran. Berbeda dengan pemasukan, pengeluaran tidak mewajibkan pilihan Anggota.
  • Mencetak & Bagikan Struk: Setelah menyimpan transaksi, Modal Struk akan muncul. Anda bisa klik Cetak untuk print menggunakan printer Thermal (Bluetooth/USB) atau klik Bagikan untuk mengirim gambar struk via WhatsApp.

2 Panduan Modifikasi & Branding

  • Ganti Nama & Warna: Masuk ke aplikasi, klik menu Pengaturan. Anda bisa mengganti nama "KasPro" menjadi nama brand Anda sendiri, serta mengganti Warna Utama sesuka hati.
  • Mengatur Icon PWA (PENTING!): Agar aplikasi bisa diinstal di HP/Desktop dengan logo buatan Anda, lakukan ini:
    1. Buat folder bernama assets di sebelah file index.html.
    2. Siapkan gambar logo Anda (harus PNG tanpa background rumit).
    3. Buat 2 ukuran gambar tersebut: 192x192 pixel dan 512x512 pixel.
    4. Simpan ke folder assets dengan nama masing-masing: icon-192x192.png dan icon-512x512.png.

3 Panduan Instalasi & Hosting (Upload)

Karena ini adalah Web App Vanilla JS, Anda TIDAK PERLU Node.js atau Database MySQL. Anda cukup upload file ini ke Hosting gratis atau berbayar.

Syarat Wajib PWA: Hosting/Website Anda WAJIB menggunakan HTTPS (SSL). Jika menggunakan HTTP biasa, fitur Install Aplikasi & Service Worker (Offline) tidak akan bekerja.

Cara Upload ke cPanel (Hosting Berbayar):

  • Login ke cPanel hosting Anda.
  • Buka File Manager -> masuk ke folder public_html.
  • Upload keempat file ini (index.html, manifest.json, sw.js) dan folder assets ke dalam public_html.
  • Buka domain Anda, selesai!

Cara Install ke HP (Android/iOS):

  • Buka link website Anda melalui Google Chrome (Android) atau Safari (iOS).
  • Tunggu beberapa detik, akan muncul notifikasi "Add to Home Screen / Instal Aplikasi" di bagian bawah.
  • Jika tidak muncul, klik ikon Tiga Titik di kanan atas (Chrome) atau ikon Share di bawah (Safari), lalu pilih Add to Home Screen.
  • Aplikasi sekarang sudah terinstal seperti aplikasi Native dan bisa dibuka 100% tanpa internet.
